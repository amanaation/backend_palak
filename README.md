# Steps

1. Clone the repo
2. cd into ``` main ```
3. Run the following command only on Linux/Mac:

``` python 
    python3 -m venv .venv
```
```bash
    source venv .venv/bin/activate
```

4. Install the requirements using the following command:

```bash
    pip install -r requirements.txt
```

5. Start the local development server:
``` python
    python3 main.py
```
